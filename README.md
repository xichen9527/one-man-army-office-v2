# 众包办公平台 (One-Man Army Office V2)

Next.js 16 + Supabase 驱动的现代化办公平台。

## 技术栈

- **框架**: Next.js 16.2 (App Router, Turbopack)
- **语言**: TypeScript 5
- **样式**: Tailwind CSS 4 + shadcn/ui
- **状态管理**: Zustand
- **后端**: Supabase (Auth, Database, Realtime, Edge Functions, Storage)
- **部署**: Vercel + GitHub Actions

## 功能模块

- 📊 Dashboard - 数据仪表盘
- 📋 Tasks - 任务看板（拖拽）
- 📂 Documents - 文档管理
- 🤝 Collaboration - 团队协作
- 💼 CRM - 客户管理
- 📱 Social Media - 社媒管理
- 🎬 Video Conference - 视频会议
- 🤖 AI Assistant - AI 助手
- ⚙️ Settings - 个人设置
- 🔧 Admin - 管理面板

## 开发

```bash
npm install
npm run dev      # 开发服务器
npm run build    # 生产构建
```

## 部署

推送到 GitHub 后自动通过 Vercel 部署。
